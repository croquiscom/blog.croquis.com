Biscuit 3.9.5가 릴리스 됐습니다.

2014-03-04

안녕하세요. 크로키입니다.
Biscuit이 3.9.5로 업데이트되었습니다.

3.9.2에서 다음과 같은 변경이 있었습니다.
  • 뜻 보기 시간을 설정할 수 있습니다. 1초, 2초, 4초, 8초중 하나를 선택할 수 있습니다.
  • 뜻 편집시 품사를 입력할 수 있습니다. 소프트웨어 키보드 위에 품사 입력 버튼이 있습니다.
  • 한번 사용자가 뜻을 편집한 경우, 단어를 변경해도 뜻이 유지되도록 해서 의도치않게 사용자가 입력한 뜻이 사라지는 것을 방지했습니다.
  • 영어-아랍어 사전이 추가됐습니다.

iOS 버전은 다음과 같은 문제점들이 수정됐습니다.
  • iOS 5.x에서 앱이 동작하지 않는 문제
  • 아이패드에서 이미지 단어 인식 기능을 사용하지 못하는 문제
  • 일부 환경에서 단어 알림의 시각이 0:00으로 설정되는 문제
  • '블루투스 키보드' 또는 '아이패드 분리형 키보드’ 사용시 엔터키로 편집 완료가 되지 않는 문제

안드로이드 버전은 다음과 같은 문제점들이 수정됐습니다.
  • 뜻 편집 중 홈 스크린으로 나가면 화면이 보이지 않는 문제
  • 단어 입력시 ‘붙여넣기’ 버튼을 누를 수 없는 문제
  • 목록 순서를 변경하지 못하는 문제

더욱더 발전하는 Biscuit 팀이 되겠습니다. 감사합니다.

Biscuit 3.9.2이 릴리스 됐습니다.

2013-12-19

안녕하세요. 크로키입니다.
Biscuit이 3.9.2로 업데이트되었습니다.

3.9.0에서 다음과 같은 변경이 있었습니다.
  • 비스킷 상자를 에버노트로 내보내는 기능에 더해, 에버노트에 있는 노트를 Biscuit으로 가져올 수 있는 기능이 추가되었습니다. 에버노트에서 Biscuit 노트북에 노트를 만든 후 단어를 한줄에 하나씩 쓴 후 Biscuit으로 가져오면 비스킷 상자가 만들어지고 뜻이 자동으로 채워집니다. 이후 다시 에버노트로 내보내면 예쁜 단어장이 만들어집니다.
  • 드랍박스 서비스로 비스킷 상자를 내보내거나 가져올 수 있습니다. 내보내기시 각 비스킷 상자는 하나의 텍스트 파일로 변환됩니다. 반대로 단어가 한줄에 하나씩 쓰여있는 텍스트 파일을 만든 후 가져오기를 하면 비스킷 상자가 만들어지고 뜻이 자동으로 채워집니다.
  • 영어-이탈리아어, 영어-포르투갈어 사전이 추가되었습니다. 다른 사전들도 계속 추가해 나가도록 하겠습니다.
  • 암호를 변경하는 기능이 추가되었습니다.
  • 뜻 보기 시간이 3초에서 4초로 늘어났습니다.

iOS 버전은 조금 큰 변화가 있었습니다. iPad 버전의 디자인이 큰 폭으로 바뀌었고, Bake-it이 애플 정책의 문제로 3분(iOS6 이하에서는 10분)으로 줄어들었습니다. Bake-it 시간이 줄어서 생긴 불편함은 다른 방식으로 보강해나가도록 하겠습니다.

그리고 다음과 같은 문제점들이 수정됐습니다.
  • 앱을 실행한 후 바로 단어를 편집할 때에 단어 편집이 중단되는 문제
  • 음성 기능 사용시 iCloud 백업이 되지 않는 문제
  • iOS7에서 Safari 설정에서 쿠키 차단이 되어 있으면 앱을 사용하지 못하는 문제
  • 인터넷이 안 되면 로그인화면에서 멈춰있는 문제
  • iPad에서 Flashcards의 시간을 변경할 수 없는 문제

안드로이드 버전은 기존에 iOS와 유사한 디자인에서 안드로이드 기본의 ActionBar 스타일로 변경되었습니다.

그리고 다음과 같은 문제점들이 수정됐습니다.
  • 음성 기능 사용시 음성 파일이 음악 플레이어에 노출되는 문제
  • 에버노트 연동 기능이 동작하지 않는 문제
  • 4.3 젤리빈에서 Bake-it 사용시 죽는 문제

더욱더 발전하는 Biscuit 팀이 되겠습니다. 감사합니다.
