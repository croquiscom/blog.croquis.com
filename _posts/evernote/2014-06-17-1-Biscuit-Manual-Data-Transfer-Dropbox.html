This post will cover Biscuit’s functions for exporting to and importing from Dropbox(https://www.dropbox.com/home). 


Functions with Dropbox can be accessed through the settings menu as shown in the pictures below. 




Link

You will need an account with Dropbox to connect with Biscuit, please check the url provided at the top. You will need to first link your Dropbox account; when prompted please sign in with your Dropbox account, not your Biscuit account.  




After Link




Export

When you export to Dropbox all of your Boxes will appear as text files in the Apps folder within a new folder named Biscuit. The entries within each file correspond to your Box in the following ways.
Each line is a flashcard from your Box. For each line the format is: word [tab] definition [tab] normal(0) / mark(1) / dim(2) . 




Import

When you are importing from Dropbox make sure to use a .txt file placed within the same folder that Dropxbox exports to, the Biscuit folder within the Apps folder. The name of the .txt file will become the name of the Box in Biscuit. To create or modify flashcards within your .txt files you have two options:
1) Only fill in the word portion with one word without any spaces per line. When imported, Biscuit will automatically fill in definitions that are available from your currently selected dictionary.
2) Create or modify flashcards using the following format for every line: word [tab] definition [tab] 1(for marked) / 2(for dimmed) / leave blank(for normal)


Unlink

The Unlink button that appears in the menu will sign you out of your Dropbox account and disconnect Dropbox from Biscuit. You will need to link a Dropbox account again to export and import in the future.
